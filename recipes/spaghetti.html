Description

Spaghetti is the quintessential Italian pasta. It is long like a string, round in cross-section and made from durum wheat semolina.

Ingridients

  • 100g Spaghetti
  • 200g Mince
  • Tomato Paste
  • Parmesan Cheese
  • Onion
  • Oil
  • Garlic bread (Optional)

Steps

  1. Add water to the boiling pot and start heating it up
  2. Add cooking oil to the fry pan and start heating it up
  3. When the oil is hot add onions and start frying
  4. When the onions start sizzling add all the mince and start stirring"
  5. Once the water has started boiling in the pot, place the spaghetti inside
  6. As the mince starts turning brown, add a couple of spoons of tomato paste and start stirring
